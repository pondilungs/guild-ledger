GUILD LEDGER — v0.5.6

You are NOT the hero. You are the guild accountant.

Rent adventurers, send expeditions, collect profit when they die, and grow your guild empire. Every death is a line item. Every quest is a receipt.

HOW TO PLAY
• Hire party members — passive gold every second
• Send expeditions — burst profit, death risk
• Upgrade the guild — multiply income
• Unlock 6 zones — each tier pays more than the last
• Prestige — reset for permanent reputation bonuses
• Tap Collect — 2× income boost (30s on / 30s cooldown)
• Reputation Market — spend prestige on permanent upgrades (leaderboard uses lifetime rep.)

ZONE LOOT — endgame grind
Rare materials drop while you farm each zone. Stockpile them over hours or days, then craft permanent gear that survives prestige:

• Rusty Wire → Rat Trap (+5% gold)
• Iron Shard → Goblin Pickaxe (+10% gold)
• Tomb Wax → Crypt Lantern (+8% DPS)
• Dragon Fang → Vault Key (+12% quest gold)
• Hell Ink → Infernal Stamp (+10% offline)
• Audit Wax → Audit Seal (-4% death chance)

Gear does NOT increase drop rates — it's pure long-term income.

ONLINE — fresh season (v0.5.6)
• Global leaderboard reset — new season, race again from zero
• First to 20 Rep race
• Username + auto profile sync

FEATURES
• Browser idle/incremental RPG — no download
• Turkish & English UI
• Zone loot workshop with scrollable mobile UI
• One-time patch notes when updates ship
• Fullscreen dashboard + flavor event log
• Offline earnings (8h cap), auto-save, tutorial
• Mobile-friendly

v0.5.6 — Loot workshop UI refresh, patch notes modal, scrollable workshop, fresh leaderboard season.

Feedback welcome — we're actively updating!

Controls: Mouse / touch
